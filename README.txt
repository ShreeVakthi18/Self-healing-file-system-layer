SHFSL 2.0 - Self-Healing File System Layer (Demo)
-----------------------------------------------------
This demo folder contains sample files used to test the SHFSL watcher, snapshotting, and decoy detection.

How to use:
1. Run the watcher (python watcher.py or shfsl.py) to start monitoring.
2. Modify or delete files to see snapshots and rollbacks.
3. Open dashboard.py (Streamlit) to view snapshots and alerts.

Files in this folder are sample/demo data only.
